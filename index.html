OnClick

O evento onclick é usado para executar uma determinada função ou ação quando um elemento é clicado. Por exemplo, um botão pode ter um evento onclick que executa uma função que exibe uma mensagem de alerta

ondblclick

O evento ondblclick é um evento do JavaScript que é acionado quando um elemento é clicado duas vezes com o mouse.

onChange

O evento onchange é um evento do JavaScript que é acionado quando o valor de um elemento HTML é alterado.

MouseOut

O evento onmouseout é um evento JavaScript que é acionado quando o usuário move o cursor do mouse fora de um elemento HTML.
